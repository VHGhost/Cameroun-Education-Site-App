# CamerEdu — Site éducatif pour les terminales et les étudiants mal orientés au Cameroun

Ce projet est un site statique simple conçu pour les élèves de terminale et les étudiants universitaires mal orientés au Cameroun.

## Contenu
- `index.html` : page principale du site
- `styles.css` : styles visuels et responsive
- `script.js` : interactions légères pour le menu mobile et le quiz

## Installation
1. Ouvrez le dossier `CameroonTerminaleSite`.
2. Ouvrez `index.html` dans un navigateur web.

## Personnalisation
- Ajoutez des fiches et sujets dans de nouveaux fichiers HTML ou PDF.
- Renommez `CamerEdu` si vous souhaitez un autre nom de marque.
- Intégrez des quiz plus avancés dans `script.js`.

## Préparer le dépôt GitHub
1. Installer Git si nécessaire : https://git-scm.com/download
2. Ouvrir un terminal dans le dossier `CameroonTerminaleSite`
3. Exécuter :
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   ```
4. Créer un dépôt GitHub en ligne
5. Lier le dépôt local au dépôt GitHub :
   ```bash
   git remote add origin https://github.com/<votre-utilisateur>/<nom-du-repo>.git
   git branch -M main
   git push -u origin main
   ```

## Déploiement sur Netlify
1. Crée un compte gratuit sur https://www.netlify.com
2. Connecte Netlify à ton dépôt GitHub
3. Choisis le dépôt `CameroonTerminaleSite`
4. Aucun build command requis pour HTML/CSS/JS statique
5. Clique sur `Deploy` et copie l’URL publique
